## Key Components Identified

### Design System Components (remsana/)

1. **Button** - Primary, secondary, tertiary, danger variants
2. **Card** - Basic, hoverable, clickable variants
3. **Input** - Text, email, password, number types
4. **Checkbox** - Single and group variants
5. **Radio** - Radio group component
6. **Progress** - Linear and circular progress indicators
7. **Badge** - Status badges with variants
8. **Alert** - Info, success, warning, error alerts
9. **Modal** - Dialog modal with sizes
10. **Toast** - Notification toasts

### Base UI Components (ui/)

- 50+ shadcn/ui components including:
  - Form controls (Select, Textarea, Switch)
  - Layout (Separator, Aspect Ratio, Resizable)
  - Navigation (Tabs, Breadcrumb, Navigation Menu)
  - Feedback (Dialog, Popover, Tooltip)
  - Data Display (Table, Chart, Avatar)

## Design Tokens

### Colors
- Primary: `#1C1C8B` (blue)
- Accent: `#eda51f` (gold)
- Text: `#1F2121` (dark gray)
- Background: `#f3f0fa` (light purple)

### Typography
- H1: 32px, semibold
- H2: 24px, semibold
- H3: 18px, semibold
- Body: 14px, normal

### Spacing
- Multiples of 8px (4px, 8px, 12px, 16px, 24px, 32px, 48px)

### Border Radius
- Small: 4px
- Medium: 8px (default)
- Large: 12px
- Full: 99px
